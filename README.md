# Incremental Backup Tool

A lightweight and reliable incremental backup tool written in Python.  
It tracks file changes, copies only updated files, and moves deleted files into a timestamped archive.

This project is ideal for academics, developers, and anyone who needs simple, automatic folder backup without complex configuration.
